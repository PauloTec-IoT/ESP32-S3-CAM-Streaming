# ESP32-S3 CAM Streaming

Projeto de streaming de vídeo em tempo real usando ESP32-S3.

## 🚀 Funcionalidades
- Modo Access Point
- Stream MJPEG via navegador
- Baixo consumo de CPU
- Compatível com PSRAM

## 📡 Acesso
Conecte-se ao WiFi:
ESP32_S3_CAM

Acesse:
http://192.168.4.1

## 🛠 Hardware
- ESP32-S3 com câmera (OV2640)
